[DIRECTIVE 4] - THE 10 STRATEGIES (IMPLEMENTATION LOGIC)

Pass these mathematical definitions to Antigravity for translation into Python logic:

Golden Cross (strat_01_macross.py):

Logic: SMA_50 and SMA_200. Buy if SMA_50 crosses above SMA_200. Sell/Liquidate if SMA_50 crosses below. timeframe: Daily bars.

Donchian Breakouts (strat_02_donchian.py):

Logic: Track MAX(High, 20_days) and MIN(Low, 10_days). Buy at 20-day high breakout. Sell at 10-day low.

Momentum Rotation (strat_03_rotation.py):

Logic: Universe = ['QQQ', 'SMH', 'XLK', 'SRVR']. Run cron job every Friday 15:30 EST. Calculate % change over 30 days. Liquidate current holdings, invest 100% in the ETF with highest % change.

MACD Trend (strat_04_macd.py):

Logic: Fast=12, Slow=26, Signal=9. Buy when MACD line crosses signal line from below, and MACD < 0. Sell when MACD crosses below signal line above 0.

RSI Dip (strat_05_rsi_dip.py):

Logic: 14-period RSI on 15-min bars. Buy if RSI < 30. Sell if RSI > 70.

Bollinger Reversion (strat_06_bollinger.py):

Logic: 20-period SMA, 2 StdDev. Buy when price touches lower band. Sell when price touches middle band (SMA).

VIX Filtered Reversion (strat_07_vix_filter.py):

Logic: Same as #5, but query SPY VIX index. If VIX > 30, block all buy orders (Risk off).

VWAP Bounce (strat_08_vwap.py):

Logic: Calculate daily VWAP (Intraday minute bars). Buy if current price crosses VWAP from below AND current 1m volume > average 15m volume. Close position at 15:50 EST.

Pairs Trading (strat_09_pairs.py):

Logic: Spread = Price(QQQ) / Price(XLK). Calculate Z-score of spread over 20 days. If Z > 2, Short QQQ, Long XLK. If Z < -2, Long QQQ, Short XLK. (Note: Ensure paper account has margin enabled).

Grid Trading (strat_10_grid.py):

Logic: Set baseline price. Place limit buy orders every -1% below baseline. Place limit sell orders every +1% above baseline. Track local state of filled orders to replenish grid.
